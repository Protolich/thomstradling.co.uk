My Work

Adido

HTML • CSS • JS • PHP • MODX

The Adido website was redesigned as part of a larger rebrand and strategic repositioning. I was responsible for implementing the new designs from the internal designer in HTML & CSS and using JavaScript to provide any required interactivity. This project involved close collaboration with both the designer and the brand manager to make sure the website was focussed on the intended message.

 

Bournemouth Council

HTML • CSS • C# • Razor • Contensis

As part of Bournemouth Council's plan to modernise it's digital service offerings I was responsible for producing the markup and styling for the new website. After creating the template pages I then integrated the new site into Contensis, the CMS used by the council. The integration involved implementing the static templates with Contensis and using C# and Razor Views to set up and control some of the programatically generated portions of the templates.

The project has also involved ongoing support for the main council site as well as work on several related websites such as the Council's Digital by Default platform, the Hengistbury Head website (visithengistburyhead.co.uk) and the Bournemouth Beach Lodges website (bournemouthbeachlodges.co.uk).

Charmed

HTML • CSS • JS • PHP • MODX • ECOM

For the Charmed by Michael Matthews website they needed an online ecommerce platform to support their business going forward.

The site started life in Adido's in-house CMS before being re-implmenented in MODX to ensure the site could be best supported going forward. I was responsible for implementing the designs in HTML & CSS and integrating them with the CMS.
